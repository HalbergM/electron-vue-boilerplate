# A boilerplate app using Electron & Vue.js

### Requirements
    Electron >= 1.0.0
    Vue >=2.0.1
    Vue Router >=2.0.0
    Webpack >=2.1.0-beta

## Usage

```bash
git clone https://github.com/sohelamin/electron-vue-boilerplate.git
cd electron-vue-boilerplate && npm install
```

## Run

```bash
npm run dev-server
npm run dev // on a different tab
```

## Build

```bash
npm run build
npm run package-osx
```
